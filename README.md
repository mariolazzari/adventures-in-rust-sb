## How to Start Learning Rust (for TypeScript Developers)

### Installation

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
which rustc
which cargo

rustup update
```

#### Create first project

```sh
cargo new first-rust-project
cargo build
cargo run
```

#### Tools

- Rust analyzer
- Clippy
